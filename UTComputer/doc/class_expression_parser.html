Description détaillée

Un objet ExpressionParser est un objet utilitaire de parsing d'expressions arithmétiques.

Cette classe utilise l'algorithme Shunting-yard. Il permet, étant donné une chaîne de caractères en notation infixe, de produire un ensemble d'objets Operand en notation postfixe.

std::string ExpressionParser::readToken ( )

Consume une entité dans l'expression courante.

Renvoie
Chaîne de caractère correspondant à une entité ou chaîne vide si aucun jeton n'a été trouvé.

Un jeton peut être :

  • Un nombre entier ou flottant;
  • Un symbole (i.e. caractère non chiffre et non lettre);
  • Une suite de lettres capitales et/ou de chiffres commençant par une lettre capitale.
