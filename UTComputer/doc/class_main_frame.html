Description détaillée

La classe MainFrame est un élément graphique qui contient le clavier clickable.

Le clavier clickable est constitué des chiffres, des opérateurs symbolique, d'un bouton vers la fenétre des autres opérateurs, et de touches tel que "Entré", backspace etc.
