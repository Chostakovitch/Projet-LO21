Description détaillée

Un objet ProgramLiteral représente une suite d'opérandes ordonnées.

Cette suite d'opérandes constitue un algorithme qui ne s'évalue pas en un résultat particulier. Un programme étant aussi une opérande, on peut référencer des sous-programmes. Cette classe utilise le design pattern Composite.
